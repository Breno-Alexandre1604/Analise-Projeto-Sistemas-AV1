# 🚀 Projeto APS: Análise e Desenvolvimento de Sistemas (Lista 01)

Este repositório contém a resolução detalhada de 11 problemas propostos na disciplina de **Análise e Projeto de Sistemas (APS)**. O projeto abrange desde o levantamento de requisitos e modelagem UML até a implementação funcional utilizando Python e streamlit.

## 📋 Descrição do Projeto
O objetivo deste trabalho é demonstrar a capacidade de transpor problemas de negócio para soluções computacionais estruturadas, seguindo as melhores práticas de:
1. **Engenharia de Requisitos:** Identificação de funcionalidades e restrições.
2. **Modelagem de Dados:** Criação de diagramas de classes UML (Atributos e Métodos).
3. **Desenvolvimento de Software:** Implementação de interfaces interativas e lógica de programação.

---

## 📁 Estrutura do Repositório

* **`/diagramas`**: Contém as representações visuais das classes para as 11 questões.
* **`requisitos.txt`**: Documentação técnica contendo os Requisitos Funcionais (RF) e Não Funcionais (RNF).
* **`/src`**: Código-fonte das soluções, nomeados de `questao01.py` a `questao11.py`.
* **`README.md`**: Guia principal do projeto.

---

## 🛠️ Tecnologias e Ferramentas
* **Modelagem UML**: Utilizada para representação da arquitetura do sistema.
* **Python 3.x**: Linguagem base para o desenvolvimento da lógica.
* **Streamlit**: Framework utilizado para a criação das interfaces web rápidas e responsivas.
* **VS Code**: Ambiente de desenvolvimento principal.
